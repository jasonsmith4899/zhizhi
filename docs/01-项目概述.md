# 智知 (ZhiZhi) AI知识库 - 项目概述

## 项目简介

智知(ZhiZhi)是一个面向中小企业的多租户AI知识库SaaS产品。商户上传文档后，系统自动解析、切片、向量化，通过RAG(检索增强生成)架构提供精准的智能问答服务。终端用户可通过微信小程序直接与AI客服对话。

## 核心特性

- **多租户架构**: 共享数据库 + tenant_id 隔离，一套服务多个商户
- **智能问答**: 基于RAG架构，从知识库检索相关信息并生成精准回答
- **多知识库**: 支持创建多个知识库，按业务场景分类管理
- **文档管理**: 支持PDF/TXT/MD格式文档上传，自动解析切片向量化
- **多轮对话**: 支持上下文记忆的连续对话（最近10条消息）
- **来源引用**: 回答中标注信息来源，可追溯可验证
- **三端合一**: 一套后端API服务商户管理后台(Vue3) + 微信小程序 + 平台管理

## 技术栈

| 层次 | 技术 | 说明 |
|------|------|------|
| 后端框架 | Spring Boot 3.4 | Java 17 |
| AI框架 | Spring AI 1.0.0 GA | DeepSeek Chat + SiliconFlow Embedding |
| 向量存储 | PostgreSQL 16 + pgvector | 业务数据+向量一体化，HNSW索引 |
| 文档解析 | Apache Tika | 支持PDF/TXT/MD |
| 商户管理 | Vue 3 + Element Plus + Pinia | 响应式SPA |
| 终端用户 | 微信小程序 | 原生开发 |
| 部署 | Docker Compose | 一键部署到腾讯云 |
