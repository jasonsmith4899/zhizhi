# 智知 AI知识库 - 数据库设计文档

## 数据库信息

- 数据库: PostgreSQL 16 + pgvector扩展
- 数据库名: zhizhi
- 用户: postgres
- 迁移工具: Flyway (V1-V5)

## 表结构

### 1. users - 用户表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| username | VARCHAR(100) UNIQUE | 用户名 |
| email | VARCHAR(200) UNIQUE | 邮箱 |
| password | VARCHAR | BCrypt加密密码 |
| role | VARCHAR(50) | 角色 |
| api_key | VARCHAR(64) UNIQUE | API密钥 |
| api_key_created_at | TIMESTAMP | API Key创建时间 |
| refresh_token_version | INT DEFAULT 0 | Token版本号 |
| daily_queries_used | INT DEFAULT 0 | 今日查询次数 |
| daily_queries_reset_at | TIMESTAMP | 查询计数重置时间 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 2. tenants - 租户表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| name | VARCHAR(200) | 租户名称 |
| logo | VARCHAR(500) | Logo URL |
| domain | VARCHAR(200) | 自定义域名 |
| plan | VARCHAR(50) DEFAULT 'free' | 套餐(free/basic/pro/enterprise) |
| status | VARCHAR(50) DEFAULT 'active' | 状态 |
| max_documents | INT DEFAULT 10 | 最大文档数 |
| max_daily_queries | INT DEFAULT 100 | 每日查询上限 |
| welcome_message | VARCHAR(1000) | 欢迎语 |
| prompt_template | TEXT | 提示词模板 |
| wechat_appid | VARCHAR(64) | 微信小程序AppID |
| wechat_secret | VARCHAR(128) | 微信小程序Secret |
| created_at | TIMESTAMP | 创建时间 |
| expired_at | TIMESTAMP | 过期时间 |

### 3. tenant_members - 租户成员表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| tenant_id | BIGINT FK→tenants | 租户ID |
| user_id | BIGINT FK→users | 用户ID |
| role | VARCHAR(50) DEFAULT 'member' | 角色(admin/member) |
| created_at | TIMESTAMP | 创建时间 |

约束: UNIQUE(tenant_id, user_id)

### 4. knowledge_bases - 知识库表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| user_id | BIGINT FK→users | 创建者 |
| tenant_id | BIGINT FK→tenants | 所属租户 |
| name | VARCHAR(200) | 知识库名称 |
| description | VARCHAR(1000) | 描述 |
| system_prompt | TEXT | 系统提示词 |
| status | VARCHAR(50) | 状态 |
| document_count | INT DEFAULT 0 | 文档数量 |
| chunk_count | INT DEFAULT 0 | 切片数量 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 5. documents - 文档表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| knowledge_base_id | BIGINT FK | 所属知识库 |
| tenant_id | BIGINT FK→tenants | 所属租户 |
| filename | VARCHAR(500) | 文件名 |
| file_type | VARCHAR(20) | 文件类型(pdf/txt/md) |
| file_size | BIGINT | 文件大小 |
| content | TEXT | 文档全文(序列化时隐藏) |
| chunk_count | INT DEFAULT 0 | 切片数量 |
| status | VARCHAR(20) | 状态(processing/completed/failed) |
| error_message | VARCHAR(1000) | 错误信息 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 6. document_chunks - 文档切片表(兼pgvector向量表)

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| document_id | BIGINT FK | 所属文档 |
| knowledge_base_id | BIGINT FK | 所属知识库 |
| tenant_id | BIGINT | 所属租户 |
| chunk_index | INT | 切片序号 |
| content | TEXT | 切片文本 |
| content_length | INT | 文本长度 |
| vector_id | VARCHAR(100) | 向量ID |
| created_at | TIMESTAMP | 创建时间 |
| metadata | JSONB | 元数据(Spring AI) |
| embedding | VECTOR(1024) | 向量(BGE-M3) |

索引: HNSW索引用于相似度搜索(COSINE_DISTANCE)

### 7. conversations - 对话表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| session_id | VARCHAR(64) UNIQUE | 会话ID |
| user_id | BIGINT FK→users | 用户ID |
| knowledge_base_id | BIGINT FK | 关联知识库 |
| tenant_id | BIGINT FK→tenants | 所属租户 |
| channel | VARCHAR(50) DEFAULT 'web' | 渠道(web/miniprogram) |
| title | VARCHAR(500) | 对话标题 |
| message_count | INT DEFAULT 0 | 消息数量 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 8. messages - 消息表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| conversation_id | BIGINT FK | 所属对话 |
| tenant_id | BIGINT FK→tenants | 所属租户 |
| role | VARCHAR(20) | 角色(user/assistant) |
| content | TEXT | 消息内容 |
| source_documents | TEXT | 来源引用(JSON) |
| token_count | INT | Token数量 |
| created_at | TIMESTAMP | 创建时间 |

### 9. visitor_stats - 访客统计表

| 列名 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| tenant_id | BIGINT FK→tenants | 租户ID |
| date | DATE | 日期 |
| total_visitors | INT DEFAULT 0 | 访客数 |
| total_queries | INT DEFAULT 0 | 查询数 |
| created_at | TIMESTAMP | 创建时间 |

约束: UNIQUE(tenant_id, date)

## 索引

### 性能索引 (V5迁移)
- idx_conversation_tenant_session: conversations(tenant_id, session_id)
- idx_message_tenant_conversation: messages(tenant_id, conversation_id, created_at)
- idx_document_tenant_kb: documents(tenant_id, knowledge_base_id, created_at DESC)
- idx_document_chunk_tenant: document_chunks(metadata->>'tenant_id')

### 其他索引
- idx_chunk_doc_id: document_chunks(document_id)
- idx_chunk_kb_id: document_chunks(knowledge_base_id)
- idx_chunk_tenant: document_chunks(tenant_id)
- idx_conv_session_id: conversations(session_id)
- idx_conv_tenant_id: conversations(tenant_id)
- idx_doc_kb_id: documents(knowledge_base_id)
- idx_doc_status: documents(status)
- idx_kb_tenant_id: knowledge_bases(tenant_id)
- idx_msg_conv_id: messages(conversation_id)
- idx_users_api_key: users(api_key)
- idx_vector_embedding: HNSW向量索引

## Spring AI专用表
- spring_ai_chat_memory: Spring AI对话记忆
- vector_store: Spring AI默认向量表(已迁移到document_chunks)
