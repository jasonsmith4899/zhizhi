# 智知 AI知识库 - 功能清单与待办

## 已实现功能

### 认证系统
- 用户注册（用户名+邮箱+密码）
- 用户登录（JWT Access Token + Refresh Token）
- Token刷新（限流5次/分钟，token版本号校验）
- API Key管理（生成/查询/吊销）
- BCrypt密码加密

### 知识库管理
- 知识库CRUD（创建/列表/详情/更新/删除）
- 系统提示词配置
- 预设提示词模板
- 删除时级联清理（文档→切片→向量→对话→消息）

### 文档管理
- 文档上传（PDF/TXT/MD，最大20MB）
- 自动解析（Apache Tika）
- 智能切片（500字/100字重叠）
- 自动向量化（SiliconFlow BGE-M3，1024维）
- 文档列表（不含全文内容）
- 文档预览/下载
- 向量状态查询
- 重新向量化
- 批量删除

### 对话系统
- 同步对话（POST /chat）
- SSE流式对话（POST /chat/stream）
- RAG检索增强（TopK=5，阈值0.7）
- 多轮对话记忆（最近10条消息）
- 来源引用（文档名+相似度+原文片段）
- 对话历史列表
- 消息详情查看
- 对话删除

### 多租户
- 租户创建/管理
- 成员邀请/角色管理(admin/member)
- TenantContext线程变量隔离
- 所有查询自动过滤tenant_id
- 向量搜索metadata过滤

### 微信小程序
- 微信授权登录
- 首页知识库列表
- AI对话界面
- 对话历史
- 个人中心

### 商户管理后台(Vue3)
- 仪表盘（知识库数、对话数、套餐）
- 知识库管理（CRUD、提示词配置）
- 文档管理（上传、预览、向量状态、批量操作）
- AI对话（流式、来源引用、文档预览）
- 对话历史（列表、消息详情）
- 个人设置（API Key管理）
- 登录/注册

### 安全与性能
- JWT认证 + API Key认证
- 接口限流（Caffeine + AOP注解）
- CORS跨域配置
- 全局异常处理
- DTO隔离（隐藏内部字段）
- 性能索引（V5迁移）
- ChatClient缓存复用

### 前端架构
- Chat组件拆分（5个子组件 + composable）
- SSE AbortController + 60秒超时
- Token刷新同步authStore
- 路由守卫（JWT exp检查）
- localStorage限制50条消息
- 跨标签页同步
- TypeScript类型系统
